# Office Electrical BIM Revit Design

## Overview
This project presents a complete electrical BIM design for a small technology consulting office workspace developed using Autodesk Revit as part of Smart Buildings and Green Technology BIM Assignment.

The project includes:

- Electrical power outlet placement
- Lighting fixture placement
- Light-switch connectivity
- Distribution board (DB) circuiting
- Receptacle connectivity
- Conduit design
- Panel schedule generation
- Electrical documentation sheets

---

## Project Scope
The office layout includes:

- Reception Area
- Open Office Workspace
- Manager Room
- Meeting Room
- Pantry Area
- Bathroom
- Server / Printer Room

---

## Key Features

### Electrical Design
- Lighting circuits connected to DB
- Power circuits connected to DB
- Switch-to-light connectivity
- Dedicated receptacles for equipment
- Pantry appliance power outlets
- Server room power connections

### Documentation
- Power Circuit Layout
